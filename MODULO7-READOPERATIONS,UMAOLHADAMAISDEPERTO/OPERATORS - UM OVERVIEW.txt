'QUERY OPERATORS'

'PROJECTION OPERATORS'...

--> PROJECTION OPERATORS ---> NOS DEIXAM '''TRANSFORM''', OU __ CHANGE__ A DATA QUE NÓS 'GET BACK' (

    pegar só alguns fields,


    ou então alterar alguns fields,

    até certo ponto...
)

'ARRAYS'... --------> E HÁ DIFERENTES TECHNIQUES 




__ PARA ___ OPERATE ESSAS COISAS... (

    NORMAL FIELDS ou arrays.... --> VEREMOS ISSO NO MÓDULO DE 'UPDATE'...
)

A PRÓXIMA PERGUNTA É:










'''COMO OPERATORS IMPACTAM NOSSA DATA'....











--> BEM,

A RESPOSTA A ESSA PERGUNTA _ DEPENDE__ DO TYPE DO OPERATOR....


















1o type) ----> QUERY OPERATOR... --> É COMO O QUE USAREMOS NESTE MÓDULO.... -->  NÓS PODEMOS USAR OS 'QUERY OPERATOR' PARA 'LOCATE DATA'... (ESSE É SEU PURPOSE)...




--> ISSO SIGNIFICA QUE OS QUERY OPERATORS_ _ NUNCA _ FARÃO O 'CHANGE' de nossa data,



E SIM 



VÃO SÓ A RETORNAR... ------> 






COMO EXEMPLO DE 'query operator',


TEMOS 



'$eq',



QUE SIGNIFICA 'equal',




OU ENTÃO 



'$gt' e '$lt',

QUE VIMOS ANTERIORMENTE...














--> 2o TYPE --> PROJECTION OPERATORS -------> VEREMOS NESSE MÓDULO, TAMBÉM... --> O QUE ESSES OPERATORS FAZEM É ALTERAR A 'DATA PRESENTATION' da data que vocÊ recebe de volta...


--> ISSO SIGNIFICA QUE ESSE OPERATOR  __TAMBÉM NÃO VAI ALTERAR A DATA NA SUA DATABASE,



E SIM __ SÓ VAI MUDAR O 'LOOK' DELA, LOOK QUE VOCê RECEBE....










--> COMO EXEMPLO DE 'Projection operators',


O PROFESSOR SÓ ESCREVE '$' ------> ESSE '$' VAI FAZER SENTIDO QUANDO COMEÇARMOS A USAR 'PROJECTION OPERATORS'....




-_> ISSO QUER DIZER QUE RECEBEMOS ALGUMAS TOOLS PARA 'CONTROL THE LOOK OF THE DATA WE GET BACK'..

















--> OK....

TEMOS 'UPDATE OPERATORS'...













3o type --> UPDATE OPERATORS --> TE DEIXAM 'MODIFY + ADD' ADDITIONAL DATA... ---> ESSE TIPO DE OPERATORS, NO CASO, VAI TE DEIXAR/VAI OBRIGATORIAMENTE 'CHANGE DATA'...









--> COMO EXEMPLO DE 'update operator,'



PODEMOS CITAR 


'$inc' ---> 





ESSE OPERATOR DE '$inc '

'''INCREMENTAR UM 


FIELD POR 1 NÚMERO, OU QUALQUER NÚMERO QUE VOCÊ ESPECIFICAR, QUANDO 


ESSE DOCUMENT FOR ALTERADO'''..... ------> E VEREMOS ISSO NO MÓDULO DE 'UPDATE'....
